# The Carey Network

A multi-page dashboard website built with Next.js and a dark orange theme.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Main landing page with welcome section and quick access links |
| `/apps` | Apps dashboard with horizontal row of large app icons (YouTube, Discord, Gemini, Twitch, Snapchat) |
| `/games` | 6-column responsive grid of game icon placeholders |
| `/settings` | 2x2 grid with Cloak, Auto Cloak, Panic Button, and Web-App Mode |
| `/extras` | Additional features placeholder |
| `/dashboard` | Dashboard configurations (coming soon) |
| `/assets` | Custom assets storage (coming soon) |
| `/images` | Image gallery (coming soon) |

## Features

### Settings
- **Cloak**: Disguises the tab as Google (changes title and favicon)
- **Auto Cloak**: Automatically cloaks on page load (persisted in localStorage)
- **Panic Button**: Press \` or Esc to instantly redirect to Google
- **Web-App Mode**: Optimizes display for standalone PWA experience

### Design
- Dark theme with orange gradient accents
- Floating pill-shaped navigation dock with glow effect
- Responsive layout for all screen sizes
- PWA-ready with manifest.json

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Custom components with shadcn/ui base
- **Fonts**: Geist Sans & Geist Mono
